# SportSee : sports coaching

This project creates a dashboard that aims for the user to follow all informations about his/her session, calories burned etc.

## Technologies

- Javascript
- React
- CSS Sass
- Recharts

- React-dom (v 18.2.0)

- Axios (v 0.27.2)
- Prop-types (v 15.8.1)
- Recharts (v 2.1.12)
- node-Sass (v 7.0.1)
